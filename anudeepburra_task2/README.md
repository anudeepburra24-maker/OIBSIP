# Customer Segmentation Analysis

## Oasis Infobyte Data Analytics Internship - Task 2

### Project Overview
This project focuses on customer segmentation using data analytics and machine learning techniques. The objective is to analyze customer purchasing behavior and group customers into distinct segments using the K-Means Clustering algorithm. The results help businesses understand different customer groups and develop targeted marketing strategies.

---

## Objectives

- Load and explore the customer dataset.
- Perform data cleaning and preprocessing.
- Conduct Exploratory Data Analysis (EDA).
- Calculate descriptive statistics.
- Apply K-Means clustering for customer segmentation.
- Visualize customer clusters.
- Generate business insights and recommendations.

---

## Dataset

The dataset contains the following features:

- CustomerID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1–100)

## Key Insights

- Customers were successfully grouped into distinct clusters based on annual income and spending score.
- High-income, high-spending customers represent valuable business opportunities.
- High-income, low-spending customers can be targeted with personalized marketing campaigns.
- Low-income, high-spending customers respond well to discounts and loyalty programs.
- Customer segmentation helps businesses improve marketing efficiency and customer satisfaction.
