
# House Price Prediction using Linear Regression

## Oasis Infobyte Data Analytics Internship - Level 2 Task 1

### Project Overview

This project aims to build a machine learning model using **Linear Regression** to predict house prices based on various housing features. The project demonstrates the complete machine learning workflow, including data preprocessing, feature engineering, model training, evaluation, and visualization.

---

## Objectives

* Load and explore the housing dataset.
* Perform data cleaning and preprocessing.
* Convert categorical variables into numerical format.
* Select relevant features for prediction.
* Train a Linear Regression model.
* Evaluate model performance using Mean Squared Error (MSE) and R² Score.
* Visualize the relationship between actual and predicted house prices.

---

## Dataset

**Dataset:** Housing.csv

The dataset contains information about residential properties with features such as:

* Price (Target Variable)
* Area
* Number of Bedrooms
* Number of Bathrooms
* Number of Stories
* Main Road Access
* Guest Room
* Basement
* Hot Water Heating
* Air Conditioning
* Parking
* Preferred Area
* Furnishing Status

The model performance was evaluated using:

* Mean Squared Error (MSE)
* R² Score (Coefficient of Determination)

## Key Insights

* House area has a significant impact on price.
* Air conditioning, parking, and preferred location positively influence house prices.
* Larger houses with more bedrooms and bathrooms generally have higher market values.
* The Linear Regression model effectively captures the relationship between housing features and price.

---

## Business Recommendations

* Focus on increasing valuable property features such as parking and air conditioning.
* Builders can use predictive models to estimate market prices before listing properties.
* Buyers can compare predicted prices with actual prices to make informed decisions.
* Real estate companies can integrate machine learning models into pricing systems for better valuation.
